1. 创建新的对象 ​

方法说明
Object.assign()通过复制一个或多个对象来创建一个新的对象。
Object.create()使用指定的原型对象和属性创建一个新对象。
Object.fromEntries方法把键值对列表转换为一个对象。

1.1 Object.assgin() ​

举例说明 ​

javascript
let obj_1 = { a: 1, b: 2 };
let obj_2 = { c: 3, d: 4 };
let obj = Object.assgin({}, obj_1, obj_2);

1.3 Object.fromEntries() ​

将 Object.entries 生成的键值对数组转换为对象，例如：

javascript
const map = new Map([
  ["foo", "bar"],
  ["baz", 42],
]);
const obj = Object.fromEntries(map);
console.log(obj); // { foo: "bar", baz: 42 }

2. 创建新的数组 ​

方法说明
Object.entries()返回给定对象自身可枚举属性的 [key, value] 数组。
Object.keys()返回一个包含所有给定对象自身可枚举属性名称的数组。
Object.values()返回给定对象自身可枚举值的数组。

这三种静态方法返回数组类型对象，均涉及到与array、map、set 等可枚举数据类型的互换和骚操作

将Object转换为map ​

map数据结构为[[key,value],...,[key,value]]，恰好是Object.entries返回的数组结构

javascript
var obj = { foo: "bar", baz: 42 };
var map = new Map(Object.entries(obj));
console.log(map); // Map { foo: "bar", baz: 42 }
