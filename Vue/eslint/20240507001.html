组件名称 ESLint 检查报错 ​

使用 npm create vue@latest 创建的项目在打开 ESLint 的时候，Vue 官方默认 3.X 版本的组件名称不能是 index.vue 这类单名称，报错信息如下

txt
Component name "index" should always be multi-word.
1

如果对此感觉不舒服，只需要在 .eslintrc 或 .eslintrc.cjs 文件中配置对应的 ignore 即可不检查

js
{ rules: {
    'vue/multi-word-component-names': ['off']
  }
}
1
2
3
4
